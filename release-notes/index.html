Release notes

Overview

We are continually improving the Red Hat digital design system. The following changes are listed by the date they were released. The changes will be available in the next major release following the listed date.

Design kit v1.8.0

Released on June 14, 2021

Added

External link icon

  • Added light and dark theme icons

Form

  • Added dark theme Form with light theme fields (with container, without container)

Simple navigation

  • Added bar layouts (all breakpoints)

Pagination

  • Added light and dark themes

Progress steps

  • Added a new website page for Progress steps component

Universal navigation

  • Design done, MVP component is live across redhat.com

Deprecated

Form

  • Deprecated Form with dark theme fields (waiting on PatternFly’s dark theme release)

Design kit v1.7.0

Released on March 15, 2021

Added

Breadcrumb

  • Added light and dark themes

Default hero

  • Added light and dark themes

External link icon

  • Added light and dark themes

Jump links

  • Added a horizontal layout

Link styles

  • Added light and dark themes

Primary navigation

  • Added a full tray version (all breakpoints)
  • Added 2-column and 3-column compact tray layouts (Desktop only)
  • Added bar layouts with a logged in avatar (all breakpoints, currently redhat.com only)

Popover

  • Added light and dark themes

Stateful button

  • Added a copy to clipboard Stateful button in light and dark themes
  • Learn more about Stateful buttons

Switch

  • Added variants with and without labels

Changed

Announcement

  • Changed the large component sizes (Desktop and Tablet) to support background images

Modal

  • Changed background opacity to align 1:1 with PatternFly

Design kit v1.6.0

Released on October 6, 2020

Added

Avatar

  • Added dark theme Default avatar
  • Added dark theme Default avatar with attribution text

Color

  • Added dark theme Danger color (#FE5142)

Form

  • Added light and dark theme Form in a container
  • Added dark theme floating Form
  • Added dark theme input groups for floating Form

Label

  • Added filled and outlined Labels

Sticky card

  • Added light and dark theme Sticky cards to the Optimization patterns page

Tooltip

  • Added light theme (black) Tooltips in top, right, bottom, and left orientations
  • Added drop shadow to light theme (black) Tooltip

Typography

  • Added dark theme text styles (Desktop)
  • Added dark theme list styles (Desktop)

Changed

Block quote

  • Moved light and dark theme quote text below quote icon in all left-aligned configurations

Color

  • Changed dark theme link color for better accessibility

Interaction states

  • Changed color of the focus indicator to stay current with browsers

Naming

  • Changed standard Form to floating Form

Box Tabs

  • Changed styling of tab group to include an inset on the left

Deprecated

Global

  • Deleted empty text XD layers in all components

Accordion

  • Deleted dark theme Accordions for mobile

Alert

  • Deleted Inline alert variants because they have not been rolled out by PatternFly yet

Announcement

  • Deleted dark theme Announcement because it clashes with the Primary navigation component

Button

  • Deleted Large close button variant to align to using one size of close button everywhere like PatternFly

Call to action

  • Deleted Disabled Call to action because it is not used anywhere
